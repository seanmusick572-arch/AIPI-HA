# AIPI-HA

Custom Home Assistant Voice firmware for the XOrigin AI Pi-Lite.

## Goals

- Home Assistant Voice Assistant
- ESPHome compatible
- Wake word support
- ES8311 audio
- ST7735 LCD
- OTA updates
- Open source

## Hardware

- ESP32-S3-WROOM-1
- 16 MB Flash
- 8 MB PSRAM
- ES8311 Audio Codec
- 128x128 SPI LCD
- WS2812 RGB LED
- Push Button
